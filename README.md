# Early Disease Risk Prediction

## Project Overview
Machine Learning project that predicts cardiovascular disease risk using patient health records.

## Dataset
- 68,000+ patient records
- Features include age, BMI, blood pressure, cholesterol, glucose, smoking habits, alcohol consumption, and physical activity.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
- Matplotlib
- Seaborn

## Models Implemented
1. Logistic Regression
2. Random Forest
3. XGBoost

## Best Results
- Logistic Regression: 72.9%
- Tuned Random Forest: 73.8%
- XGBoost: 73%
